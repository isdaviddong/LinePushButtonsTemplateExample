# LinePushButtonsTemplateExample
####只是一個簡單的Line PushButton Template範例
####不做任何UI上的美化，是為了保持範例的單純性
<br/>因此，此範例只有>使用asp.net WebForm 與 .NET LineBotSDK 
<br/>請於表單中填入自己的LineBot的Channel Access Token與要傳送的UserId
<br/>按下Button即可發送 Line ButtonTemplate Message訊息
